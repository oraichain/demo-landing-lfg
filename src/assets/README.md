# Assets Directory

This directory contains video files and thumbnails for the app.

## Required Files

- `video1.mp4` - Video file for Video 1 page
- `video2.mp4` - Video file for Video 2 page
- `thumbs/video1-poster.jpg` - Poster/thumbnail image for Video 1
- `thumbs/video2-poster.jpg` - Poster/thumbnail image for Video 2
